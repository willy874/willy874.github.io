過去大概4, 5年前，當時是 bootstrap 的天下，當時最流行處理 CSS 的手段是 SASS。要嘛是 global 注入後手動管理，要嘛是採用 CSS Module，各大框架甚至有自己的一套。

近年來趨勢逐漸都開始向 Atomic 的設計靠攏，開始淘汰 Global CSS 的寫法，因為這種做法的反查難度實在太高了。

近期看到的新人對於 SASS 熟悉度越來越低，也坦白說會用到的功能越來越少。包含，迴圈、判斷、函式，而常用的變數和巢狀也隨著原生的支援使用率越來越低。

你現在都是用什麼 CSS 解決方案呢？
