這時近年前端蓬勃發展後才被提起的詞彙，前端基礎設施（Frontend Infrastructure）。隨著前端專案越來越巨大，團隊維護與協作問題逐漸浮出檯面，許多問題待優化與解決，這時候就需要強化基礎設施來改善問題。

前端架構上要密切關注的相關技術就包含以下：

- Client Services
- CI/CD tools
- Bundle tools
- Build tools
- Package management
- Server side render
- Browser render
- Frontend performance
- Frontend monitor
- Container management
- SEO
- AST
- Module Federation

隨便舉例幾個超過一半都不是一般業務開發會用到的技術，這都是得花時間去學習的。我個人也是還缺了很多拼圖，現在都還在努力去學習各項技術。

前端工程是一個發展相對晚的領域，系統架構也起步的比較晚。但隨著發展，前端應用程式不斷肥大，產品的發展與成長速度遠超架構規模。當一個 repo 有超過10個前端同時在寫功能，團隊協作效益問題就浮上檯面。

我這一年來幾乎把心思放在 Frontend Infra，太多技術高牆，太多架構面的技術問題和底層議題。你對這些技術有研究嗎？歡迎與我聊聊天分享新知唷～～
