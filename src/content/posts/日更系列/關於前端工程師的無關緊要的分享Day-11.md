去年搞了一整年的 UI Library 難產，總是把時間花在管理和人身上，導致實質自己的產出很糟，有時候真的一個人衝得快。

最近開始重新籌備 UI Library Project 的再次啟動，重新規劃創建架構，有一套比較完整的做法可以提供出來被執行。剩下我不打算花時間處理人的問題了，我打算自己按照內心所想的衝刺，願意跟上來的我再花時間。

這個題目想找到一起努力的夥伴很難，畢竟不是每個人都對這個題目有興趣。如果你喜歡這個題目，歡迎可以找我！
