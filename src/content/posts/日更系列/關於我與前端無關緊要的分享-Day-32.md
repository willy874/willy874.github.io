React 更新了，React 19.2 ！

其中我最在意的功能莫屬 **useEffectEvent**。

一直以來 react 在下傳 function 時要去掛勾 useEffect 常常造成不必要的更新，這不是用 useCallback 能解決的，也解決以往我在開發時的痛點。

其他還有 Activity, **cacheSignal** 等等API，也為 SSR 相關機制與渲染進行優化與新功能提供。
