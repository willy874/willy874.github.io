很多時候一個人努力的軌跡都會在程式碼中，他的執著和熱情是可以看到的，一點一滴的打磨中有各種痕跡。

平常幫同事、客戶、學生、朋友 Code Review ，慢慢的都可以看到一個人的成長軌跡。AI 出來前其實資訊量更大，自從 AI 的洪流到來，多了很多雜訊，為了去蕪存菁，需要花費更大的心力。但有趣的是，AI 寫的跟家去複製貼上的更好辨識，因為 AI 寫的東西就是有一股「AI 味」。

現在的新人過度依賴 AI，無法為自己撰寫的交付負責，真覺得 Code Review 的負擔是越來越大的。過去花時間在設計和嘗試，現在花更多時間在定義和測試。
