之前做的那個 UI 專案陣亡了，這次我打算自己顧自己的衝刺。初步先開了 UI Library Repo，很多東西都還沒整理好。不過打算邊做邊分享遇到的事情，來個 Build in Public 的開發，可以讓大家看著我做這個東西。

其實核心我有一個想完成的東西，但那個核心產品我覺得我想先賣個關子。

老樣子，這就是一個造輪子專案，怎麼難怎麼來。我打算基於 zag.js, panda.css, oxc lint 等等技術線為基礎來製作，大致上會參考 Chakra UI 相關專案的架構來進行設計。有不錯的實作我也會考慮直接抄，然後微調成我的形狀。

如果有興趣參與 UI Library 專案的歡迎私訊我
