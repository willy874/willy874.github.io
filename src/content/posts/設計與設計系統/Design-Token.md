這是一個近年流行的一個前端設計之間的溝通語言，雖然不算普及，但大部分會理解為變數的應用。

雖然早在 sass 流行時期已經被廣泛應用在各種 UI Framework 上面，但實際上被應用的比率依舊不高。

Design token 帶來最大的價值是對於應用的 CSS 實施規範化的共用，讓這件事不單單是開發的事，更讓設計師參與了開發流程。相對的代價，就是知識的門檻，沒有一定的知識量沒辦法把控這套系統。

Design Token 按等級區分排列有 System Token, Seed Token, Alias Token 三個大項，依照團隊需要會再變種。
