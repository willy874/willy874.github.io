我今年大概大部分的心力都在這，年初時開了這個 side project 一路做到現在，總算是看到了點成果。為了讓一群相互不認識的願意貢獻參與不是容易事，畢竟讓大家下班後努力付出不是容易事。

這個計畫你可以當作從頭建置一個 antd, mui, vuetify, element-ui 這樣的 UI Lirary。其中我並不會基於什麼 UI 框架去打造，而是「造輪子」從頭打造，從基礎設施自定義 bundle script，到每個元件細節的功能完成都嘗試去製作。別問我現在那麼多 Library 為什麼還要再造輪子？如果要說理由，那什麼也不用做了！

其實 UI 設計師相對容易找，我找到很多很棒的 UI 設計師參與，許多都是在國外的台灣人，一個比一個卷。我現在仍然在找前端，能夠參與的前端工程師真的是少之又少⋯⋯

好啦，講這麼多，就是來 threads 上面找戰友，歡迎私訊我詢問詳細訊息！
