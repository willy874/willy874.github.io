最近暫停了 UI Library 的專案，覺得一群人走的好疲憊，我想一個人做。重新回到自己火力全開的開發，琢磨各種架構和 UI 元件底層。

說到開發 UI 元件，從基礎元件來說肯定就是 Button 了！很基礎，應該⋯⋯很簡單吧？

但基於 Design Token 去寫才發現，Variants, Theme, Size, Shape … 等等參數寫一波，才發現 Token 的關聯性、色彩的平衡感、權重配置ㄉ等等⋯⋯

真的不是普通的麻煩，超級複雜啊啊啊！
